# Data Science Kurs – Arbeiten mit Pandas & Scikit-Learn

Willkommen im Repository für den Data-Science-Kurs. Dieses Repo begleitet dich durch alle zentralen Schritte moderner Datenanalyse – von der ersten Dateninspektion über saubere Datenaufbereitung bis hin zu Machine-Learning-Modellen (supervised & unsupervised).

**Der Fokus liegt auf praktischem Arbeiten, klaren Beispielen und einem schrittweisen Kompetenzaufbau.**

## Lernziele

Nach Abschluss des Kurses kannst du:

* Daten mit Pandas effizient laden, transformieren, bereinigen und analysieren
* Explorative Datenanalyse (EDA) durchführen
* Machine-Learning-Modelle mit scikit-learn erstellen, trainieren und evaluieren
* Zwischen supervised und unsupervised Learning unterscheiden
* Clustering, Klassifikation und Regression praktisch anwenden
* Reproduzierbare Data-Science-Workflows entwickeln

```text
/data/                Beispiel-Datensätze
/notebooks/           Jupyter Notebooks für jede Einheit
/output/              Output-Files aus unseren Analysen
/README.md            Kursübersicht
```

## Verwendete Technologien

* Python 3.x
* Pandas – Datenanalyse & -manipulation
* Matplotlib / Seaborn – Visualisierung
* scikit-learn – Machine Learning
